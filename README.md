
# Desafio Seazone

Esse desafio consiste em desenvolver uma automação que criasse um Card em um Board no Trello e que possuisse um banco de dados em planilhas desse mesmo Board que fosse
alimentado de forma automática com o passar do tempo.

## Dependências

- Axios

## Instalação

Após clonar o repositório execute o comando no terminal para instalar as dependências.

```bash
  npm install
```
